To run the Palindrome Program it needs 4 Inputs
first Input :: total , size of the Palindrome 
Second Input : 'w' - alphabets,
 how many alphabets you need in your Palindrome
 
Third Input :: 'n' numbers, how many numbers you need in your Palindrome
 
Fourth Input :: 's' special characters, how many special characters you need

Given Condition :: Total should not be more than 20

Provide input as :: Enter total, words, numbers and special chars in a single line separate by space, FOR eg :: 8 3 5 4
